nodex.pw Media Editor 3.2.5

Локальный медиаредактор для Windows. Electron приложение без серверных зависимостей.

Что умеет

Многодорожечный аудиоредактор с отображением формы волны. Плавный кроссфейд между треками для создания миксов. Встроенный конвертор форматов через ffmpeg. Редактор обложек с поддержкой анимированных gifки и mp4. Скачивание треков с AppleMusic и SoundCloud через ytdlp. Браузер ассетов со сканированием файлов на ПК. Проекты в формате .nodexproj открываются двойным кликом.

Технологии

Electron | HTML | CSS | JavaScript | FFmpeg | yt-dlp

FFmpeg и ytdlp скачиваются автоматически в AppData при первом запуске если не найдены.

Сборка

npm install
npm run build

Выходной файл появится в папке dist.
